# Arduino IDE en el proyecto

Arduino IDE es el entorno utilizado para editar, compilar y transferir el programa al ESP32-S3.

## Función dentro del proyecto

Permite:

- editar el sketch principal;
- seleccionar la placa correcta;
- verificar errores de compilación;
- gestionar bibliotecas;
- abrir el monitor serial para depuración.

## Flujo básico

1. Instalar Arduino IDE 2.x.
2. Instalar el soporte de placas ESP32.
3. Seleccionar la placa ESP32-S3.
4. Elegir el puerto disponible.
5. Verificar el sketch.
6. Transferir el programa al dispositivo.
7. Revisar mensajes de arranque en el monitor serial.

## Ventajas

- rapidez en prototipado;
- integración simple de bibliotecas;
- entorno adecuado para pruebas académicas;
- depuración rápida por puerto serial.
