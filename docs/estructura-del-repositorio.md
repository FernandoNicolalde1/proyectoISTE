# Estructura del repositorio

La organización actual del repositorio separa documentación, recursos y código de referencia.

## Carpetas

### `docs/`
Documentación técnica y descriptiva del proyecto.

### `src/`
Espacio previsto para sketches o código fuente propio del proyecto.

### `lib/`
Reservado para componentes auxiliares propios.

### `assets/`
Espacio para diagramas, capturas, logotipos y otros recursos visuales.

## Criterios aplicados

- separar documentación de código;
- mantener una base clara para revisión académica;
- facilitar futuras ampliaciones del proyecto.
